文档

使用说明和 API 参考

快速开始

# 克隆仓库
git clone https://github.com/yourusername/IMCs.git
cd IMCs

# 安装依赖
pip install -r requirements.txt

# 运行 Web UI
streamlit run src/ui/app.py

Python API

from src.agents.core import MLAgent, TheoryDataAgent

# 初始化 ML 智能体
agent = MLAgent()

# 加载数据并训练
X, y = agent.load_data_from_cifs("data/theory/cifs", "data/theory/formation_energy_full.json")
top3 = agent.train_all(X, y)

# 最佳模型: XGBoost (R² = 0.90)
